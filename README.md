# 3D Modeling & Blender Scripting

Projekty zaměřené na 3D modelování, přípravu modelů pro 3D tisk a automatizaci modelování v Blenderu pomocí Python API.

## 🛠️ Obsah projektu
- **Python skripty pro Blender (`shark_blender.py`, `zralok_detailni.py`):** Skripty využívající Blender API (`bpy`) k procedurálnímu generování a detailní úpravě 3D modelů žraloků (nastavení částic, geometrie, modifikátorů).
- **3D modely pro tisk (.stl):**
  - `articulated_shark.stl` – pohyblivý model žraloka (flexi).
  - `funny_octopus.stl` / `flexi_octopus.stl` – modely chobotnic.
- **Zdrojové soubory Blenderu (.blend):** Kompletní pracovní scéna.

## 💻 Použité technologie
- **Python** – Blender API scripting (`bpy`).
- **Blender** – 3D modelování, renderování, částicové systémy.
- **STL / 3D tisk** – příprava modelů pro FDM tiskárny.
